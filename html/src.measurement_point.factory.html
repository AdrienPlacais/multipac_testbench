src.measurement_point.factory module

Define a class to create the proper IMeasurementPoint.

class IMeasurementPointFactory(**kwargs)

Bases: object

Class to create the proper GlobalDiagnostics PickUp.

It infers the proper type, position of instruments as well as the measured data from the configuration .toml file and the measurements .csv file.

run_single(config_key: str, config_value: dict, df_data: DataFrame) → IMeasurementPoint

Create a single measurement point.

Parameters:

  • config_key (str) – A key from the .toml file. If ‘global’ keyword is in the key, we return a GlobalDiagnostics. Else, we return a PickUp.

  • config_value (dict) – Values from the .toml file corresponding to config_key, which will passed down to the created IMeasurementPoint.

  • df_data (pd.DataFrame) – Full data from the .csv file.

Returns:

A GlobalDiagnostics or PickUp.

Return type:

IMeasurementPoint
