src.instruments.powers module

Define power probe to measure forward and reflected power.

Also include calculation of reflection coefficient \(\Gamma\) and (voltage) SWR \(SWR\).

class Powers(*args, **kwargs)

Bases: Instrument

An instrument to measure forward and reflected powers.

classmethod ylabel() → str

Label used for plots.

property gamma: ndarray

Return the reflection coefficient.

Reflection coefficient gamma is defined as:

\[\Gamma = \frac{V_r}{V_f} = \sqrt{\frac{P_r}{P_f}}\]

where \(_f\) means forward and \(_r\) reflected.

property swr: ndarray

Give the voltage Standing Wave Ratio.

\[SWR = \frac{1 + |\Gamma|}{1 - |\Gamma|}\]
property forward: ndarray

Return forward power only.

property reflected: ndarray

Return reflected power only.

where_is_growing(**kwargs) → list[bool | float]

Determine where power is growing (True) and where it is not.
