data_for_somersalo_scaling_law(multipactor_bands: TestMultipactorBands | InstrumentMultipactorBands, use_theoretical_r: bool = False, **kwargs) → DataFrame

Get the data necessary to plot the Somersalo scaling law.

In particular, the power thresholds measured during the last half power cycle, and the reflection coefficient \(R\) at the corresponding time steps. Lower and upper thresholds are returned, even if Somersalo scaling law does not concern the upper threshold.

Parameters:

  • multipactor_bands (TestMultipactorBands | InstrumentMultipactorBands) – Object telling where multipactor happens. If it is a TestMultipactorBands, we merge all the InstrumentMultipactorBands in it, to know where the first (several_bands_politics='keep_first') multipactor happened, anywhere in the testbench (union='relaxed'). You can also provide directly an InstrumentMultipactorBands; we will take its last MultipactorBand.

  • use_theoretical_r (bool, optional) – If set to True, we return the \(R\) corresponding to the user-defined \(SWR\). The default is False.

  • kwargs – Other keyword arguments passed to at_last_threshold().

Returns:

Holds the lower and upper \(P_f\) during last half power cycle, as well as reflection coefficient \(R\) at same time steps.

Return type:

pd.DataFrame
