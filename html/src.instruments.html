This is useful to easily match the data from a field probe to the multipactor bands measured by the curret probe at the same position.

Parameters:

  • tol (float, optional) – Mismatch allowed between positions. The default is 1e-10.

fit_voltage(full_output: bool = True) → None

Find out the proper voltage parameters.

Idea is the following: for every sample index we know the forward (injected) power \(P_f\), \(\Gamma\), and \(V_\mathrm{coax}\) at several pick-ups. We try to find \(\psi_0\) to verify:

\[|V_\mathrm{coax}(z)| = 2\sqrt{P_f Z} \sqrt{1 + |\Gamma|^2 + 2|\Gamma| \cos{(2\beta z + \psi_0)}}\]

class ReflectionCoefficient(name: str, raw_data: Series, **kwargs)

Bases: VirtualInstrument

Store the reflection coefficient.

We use the definition:

\[R = \frac{V_r}{V_f} = \sqrt{\frac{P_r}{P_f}}\]

where \(P_r\) is the reflected power and \(P_f\) is the forward power. This object is created by InstrumentFactory.run_virtual() when there is one ForwardPower and one ReflectedPower in its instruments argument.

class SWR(name: str, raw_data: Series, **kwargs)

Bases: VirtualInstrument

Store the Standing Wave Ratio.

We use the definition:

\[SWR = \frac{1 + R}{1 - R}\]

where \(R\) is the reflection coefficient.

This object is created by InstrumentFactory.run_virtual() when there is one ForwardPower and one ReflectedPower in its instruments argument.
