src.multipactor_band.multipactor_band module

Keep track of a single multipactor band.

i.e.: a set of measurement points where multipactor happens. A MultipactorBand is defined on half a power cycle.

class MultipactorBand(first_index: int, last_index: int, reached_second_threshold: bool, power_grows: bool)

Bases: object

First and last index of a multipactor zone.

multipactor_to_list_of_mp_band(multipactor: ndarray[bool_], power_is_growing: ndarray[bool_], info: str = '', several_bands_politics: str = 'merge') → list[MultipactorBand | None]

Create the different MultipactorBand.

Parameters:

  • multipactor (np.ndarray[np.bool_]) – True means multipactor, False no multipactor.

  • power_is_growing (np.ndarray[float]) – True means power is growing, False it is decreasing.

  • info (str) – To give more meaning to the error messages.

  • several_bands_politics ({'keep_first', 'keep_last', 'keep_all', 'merge'}) –

    What to to when several multipactor bands are found in the same half-power cycle:

    • 'keep_first': we keep first MultipactorBand

    • 'keep_last': we keep last MultipactorBand

    • 'keep_all': we keep all MultipactorBand (currently not implemented)

    • 'merge': the final MultipactorBand spans from start of first MultipactorBand to end of last.

Returns:

One object per half power cycle (i.e. one object for power growth, one for power decrease). None means that no multipactor was detected.

Return type:

list[MultipactorBand | None]
