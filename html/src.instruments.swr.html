src.instruments.swr module

Define a Signal Wave Ratio probe.

class SWR(*args, **kwargs)

Bases: VirtualInstrument

An object to study the SWR.

Here, SWR stands for Standing Wave Ratio and is the Voltage Standing Wave Ratio.

Deprecated since version v1.1.0.

classmethod from_rf_power_probes(power_forward: RfPower, power_reflected: RfPower, name: str = 'SWR (calculated from power probes)', tol: float = 0.05, **kwargs) → Self

Instantiate the object by manual calculation.

First, we compute the reflection coefficient, defined as:

\[\Gamma = \frac{V_r}{V_f} = \sqrt{\frac{P_r}{P_f}}\]

the SWR is then defined as:

\[SWR = \frac{1 + |\Gamma|}{1 - |\Gamma|}\]

In our case, \(P_r\) and \(P_f\) are real and positive.

Parameters:

  • power_forward (RfPower) – Holds injected rf power \(P_f\).

  • power_reflected (RfPower) – Holds reflected power \(P_r\).

  • name (str, optional) – Name of the virtual instrument.

  • tol (float, optional) – To remove \(\Gamma\) values that are too close to unity.

  • kwargs – Other keyword arguments.

Returns:

A SWR VirtualInstrument.

Return type:

SWR
