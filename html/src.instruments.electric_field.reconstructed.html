fit_voltage() → None

Find out the proper voltage parameters.

Idea is the following: for every sample index we know the forward (injected) power \(P_f\), \(\Gamma\), and \(V_\mathrm{coax}\) at several pick-ups. We try to find \(k\) and \(\psi_0\) to verify:

\[|V_\mathrm{coax}(z)| = k\sqrt{P_f} \sqrt{1 + |\Gamma|^2 + 2|\Gamma| \cos{(2\beta z + \psi_0)}}\]
voltage_vs_position(pos: float, v_f: float, gamma: float, beta: float, psi_0: float) → float
voltage_vs_position(pos: ndarray, v_f: float, gamma: float, beta: float, psi_0: float) → ndarray

Compute voltage in coaxial line at given position.

The equation is:

\[|V(z)| = |V_f| \sqrt{1 + |\Gamma|^2 + 2|\Gamma|\cos{(2\beta z + \psi_0)}}\]

which comes from:

\[V(z) = V_f \mathrm{e}^{-j\beta z} + \Gamma V_f \mathrm{e}^{j\beta z}\]
Parameters:

  • pos (float | np.ndarray) – \(z\) position in \(m\).

  • v_f (float) – Forward voltage \(V_f\) in \(V\).

  • gamma (float) – Voltage reflexion coefficient \(\Gamma\).

  • beta (float) – Propagation constant \(\beta\) in \(m^{-1}\).

  • psi_0 (float) – Dephasing constant \(\psi_0\).

Returns:

\(V(z)\) at proper position in \(V\).

Return type:

float | np.ndarray
